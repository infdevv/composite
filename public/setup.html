How to set it up

Go to your chat website of choice and find place this URL

https://composite-beta.seabase.xyz/v1/chat/completions

into the "Reverse Proxy" or "Proxy Endpoint" box

Then pick any model you want out of the available models list on the main page and set that as your model in the "Model" box

Get your API key (or generate it) and put it into the "API Key" field. Then you can save it and use it!
